Техническое задание:
-------------------
```
- Цель: web-приложение "Каталог книг"
- Стэк: Yii2 + MySQL
- Основные сущности:
    - Книга: название, год выпуска, описание, isbn, фото главной страницы
    - Авторы: ФИО
- Разграничение прав доступа:
    - Гость: просмотр, подписка на новые книги автора
    - Юзер: просмотр, добавление, редактирование, удаление (CRUD)
- Дополнительные требования:
    - Книга может иметь несколько авторов
    - Уведомление о поступлении книг из подписки должно отправляться на смс гостю
    - Раздел "Топ-10 авторов" за определенный год
- Сторонние сервисы:
    - https://smspilot.ru/ (к использованию ключ эмулятор для тестирования без реальной отправки SMS)
```
